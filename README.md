# nvim-config
Repo containing my current neovim configuration
## Dependencies
- Neovim Nightly
- NodeJS
- [packer.nvim](https://github.com/wbthomason/packer.nvim)
## Installation
`git clone git@github.com:robertoesteves13/nvim-config.git $HOME/.config/nvim`
